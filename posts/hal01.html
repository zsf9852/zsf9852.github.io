超声波测距

HAL STM32F103RBT6
STM32CubeMX
发布日期:   2021-08-12
更新日期:   2021-08-12
文章字数:   1.2k
阅读时长:   4 分

1.准备工作

硬件准备

1、首先准备的是正点原子的ATK-NANO 的STM32F103RBT6的开发板

2、超声波模块使用的是HC-SR04

该模块的四个引脚说明如下表：

引脚名 引脚作用
Vcc 5V供电（必须是5V）
Trig 触发控制信号输入
Echo 回响信号输出
Gnd 地

2.工作原理

超声波模块测距原理

如图，超声波模块发出超声波信号，如果前方碰到障碍物，则被障碍物反射回来，模块收到回来的超声波后，即可计算出与障碍物的距离：

HR-SC04模块工作原理

HR-SC04模块的时序图如下：

根据该时序图，可以看出一次测距的流程如下：

  • 触发信号由Trig引脚输入，10us的高电平即可触发一次测距；
  • 触发后，距离信息由Echo引脚输出，该输出信号的高电平时间与检测距离成比例；

在整个测距过程中，最重要的是Echo引脚输出信号高电平的时间，可以使用STM32硬件定时器得到时长，这个时间为超声波从发射到接收的时长，由声速计算出传播距离，再取1/2，计算出当前距离障碍物的距离。

3.CubeMX配置

配置时钟源

  • 如果选择使用外部高速时钟（HSE），则需要在System Core中配置RCC；
  • 如果使用默认内部时钟（HSI），这一步可以略过；

这里我都使用外部时钟：

配置GPIO

配置一个普通的GPIO，一个定时触发的GPIO，一个配置为输出模式，用于连接模块的Trig引脚，触发一次测距，这里我配置为PA0，另一个配置为输入模式（浮空），用于连接模块的Echo引脚，接收echo信号，这里我配置为PA1。

配置通用定时器TIM2

知识小卡片——STM32L431的定时器

STM32F103RBxx 系列有 1 个高级定时器（TIM1）, 3 个通用定时器（TIM2、TIM15、TIM16），两个基本定时器（TIM6、TIM7），还有两个低功耗定时器（LPTIM1、LPTIM2）。

STM32F103RB 的通用 TIMx (TIM2、TIM15、TIM16)定时器功能包括：

  • 16 位(TIM15,TIM16)/32 位(TIM2)向上、向下、向上/向下自动装载计数器，注意：
    TIM15、TIM16 只支持向上（递增）计数方式；
  • 16 位可编程(可以实时修改)预分频器，计数器时钟频率的分频系数为 1～65535 之间的任
    意数值。
  • 4 个独立通道（TIMx_CH1~4， 其中 TIM15 最多 2 个通道， TIM16 最多 1 个
    通道），这些通道可以用来作为：
    • 输入捕获
    • 输出比较
    • PWM 生成(边缘或中间对齐模式)
    • 单脉冲模式输出
  • 可使用外部信号控制定时器和定时器互连的同步电路。
  • 如下事件发生时产生中断/DMA：
    • 更新：计数器向上溢出/向下溢出，计数器初始化(通过软件或者内部/外部触发)
    • 触发事件(计数器启动、停止、初始化或者由内部/外部触发计数)
    • 输入捕获
    • 输出比较

知识小卡片结束啦~

接下来开始配置TIM2，首先选择TIM2，我们的目的是每1us计数一次 ，所以定时器的频率需要在1Mhz， 时钟源选择内部时钟：

接下来是对TIM2的参数设置，参照数据手册中的RCC时钟树，TIM2内部时钟来源是PCLK1 = 72Mhz，，所以预分频系数设置为72，即可得到1Mhz的计数频率。计数周期设为不变

4.程序编写

1、先在usart.c文件中对printf进行重定向

/* USER CODE BEGIN 1 */
#if 调用头文件
#include <stdio.h>

int fputc(int ch, FILE *stream)
{
    /* 堵塞判断串口是否发送完成 */
    while((USART1->SR & UART_FLAG_TXE) == RESET);

    /* 串口发送完成，将该字符发送 */
    USART1->DR = (uint8_t) ch;

    return ch;
}
#endif
/* USER CODE END 1 */

2、重新定义延时US级延时

void delayus(uint32_t nus)
{
		uint32_t i,j;
		for(i=0;i<nus;i++)
    {
			for(j=0;j<72;j++);
    }
}

3、驱动编写

void StartTrig(void)
{
	HAL_GPIO_WritePin(Trig_GPIO_Port, Trig_Pin, GPIO_PIN_SET);
	delayus(10);
	HAL_GPIO_WritePin(Trig_GPIO_Port, Trig_Pin, GPIO_PIN_RESET);
}

4、距离测量（main中的while中）

定义测量的距离变量
/* USER CODE BEGIN PV */
uint32_t Count;
double Distance;
/* USER CODE END PV */
StartTrig();//发送一个电平
__HAL_TIM_SET_COUNTER(&htim2, 0);//在运行时设置TIM计数器寄存器的值。
HAL_TIM_IC_Start(&htim2, TIM_CHANNEL_2);//开始TIM输入捕获测量
while(HAL_TIM_IC_GetState(&htim2)!=HAL_TIM_STATE_READY);//返回TIM输入捕获
Count = HAL_TIM_ReadCapturedValue(&htim2, TIM_CHANNEL_2);//从捕获比较单元读取捕获的值所需要的时间
Distance = Count*0.0001*330/2-10;
printf("Count = 0x%x\r\n",Count);
printf("Distance = %3.3fcm\r\n",Distance);
HAL_Delay(1000);
